# movetool-example
## 使用方法
 先在根目录 `npm i` or `yarn install`
 到 movetool-server 目录 `npm i` or `yarn install`
 根目录 运行 `npm run serve` and `npm run server`

### 开始体验

## 其他命令
`movetool i {path}`

将 `public/icons` 下的所以图片导入对应文件

`movetool id {path}`

给对应文件里的所有元素 加上 `v-id` `v-pos` 指令

`movetool rid {path}`

删除对应文件里的所有  `v-id`  指令
